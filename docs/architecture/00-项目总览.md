# IronClaw 项目总览

## 一句话定义

**IronClaw 是一个本地优先 (local-first) 的 AI Agent 运行时**，用 Go 语言编写，将 Claude AI 与工具系统（bash、文件、HTTP、浏览器）连接，通过通道（Telegram、TUI）暴露交互界面，所有数据持久化到 SQLite。

## 核心技术栈

| 类别 | 技术 |
|------|------|
| 语言 | Go 1.24+ |
| LLM 后端 | Claude (Anthropic SDK) |
| 数据库 | SQLite (WAL模式) + FTS5 全文搜索 |
| 终端UI | Bubble Tea + Glamour + Lipgloss |
| 消息通道 | Telegram Bot API |
| 工具协议 | MCP (Model Context Protocol) |
| CLI 框架 | Cobra |
| 配置格式 | YAML + 环境变量展开 |
| 构建标签 | `CGO_ENABLED=1 -tags fts5` |

## 两种 Agent 模式

### 1. Simple 模式 (默认)
线性循环：`系统提示 → LLM → 工具调用 → 重复`（最多 `max_iterations` 次）

### 2. Cognitive 模式
五阶段认知循环：
```
PERCEIVE → PLAN → ACT → OBSERVE → REFLECT
    ↑                                  │
    └──────── replan (if needed) ──────┘
```

## Gateway 初始化顺序（关键依赖链）

```
1. DB (SQLite)
2. Session Manager
3. Tool Registry (bash/file/http)
4. LLM Provider (Claude)
5. Agent Runtime (simple)
6. Memory Store (file-based) + Embedder + Compressor
7. Fact Extractor + Lifecycle Manager + Reflector + Compactor + Profiler
8. Cognitive Agent (if mode=cognitive)
9. RL System (if enabled + cognitive)
10. Knowledge Base + Hybrid Retriever + Reranker
11. Knowledge Graph + Entity Extractor + Graph Decay
12. Skill Manager
13. Multi-Agent Manager
14. Scheduler
```
